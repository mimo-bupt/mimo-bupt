课程名称 大规模天线阵通信基础
课程编号 		3111101331
学分-学时 		2 - 32
开课学期 		春季学期
必修/选修 		选修
授课老师 		高晖
适用专业 		学硕(全日)、学术型博士、本科直博、专硕(全日)、专硕(非全日)、全日制硕士进修生、全日制博士进修生、留学硕士生、留学博士生
先修课程 		概率论与数理统计，矩阵论
内容提要 		本课程的主要内容包括大规模天线阵通信系统的信道特征分析，收发机设计，传输性能分析及优化，典型应用及前沿应用等模块。授课方式以教授为主，小组讨论以及课程作业为辅，旨在让学生全面掌握大规模天线阵信道传播模型、信道估计、功率控制、复用/解复用和性能分析等核心技术，获取对复杂无线通信系统设计与分析的技能；了解大规模天线阵的典型应用及前沿应用，获取对无线通信最新技术趋势的了解。
Course Elements of Large Antenna Array Communications
Course No. 		3111101331
Credit-Course Hours 		2 - 32
Preparatory Course 		Probability and Statistics, Matrix theory
Course Description 		The main contents of this course include channel characteristic analysis of large-scale antenna array communication system, transceiver design, transmission performance analysis and optimization, typical applications and cutting-edge applications. The teaching method is mainly based on professors, supplemented by group discussions and course assignments, which aims to enable students to fully master the core technologies of large-scale antenna array channel propagation model, channel estimation, power control, multiplexing/demultiplexing and performance analysis, and acquire the skills of designing and analyzing complex wireless communication systems; Understand the typical and cutting-edge applications of large-scale antenna arrays, and gain an understanding of the latest technology trends of wireless communication.
